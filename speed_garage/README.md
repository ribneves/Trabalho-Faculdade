# Speed Garage

Um site responsivo para uma concessionária de carros, desenvolvido com HTML5 e CSS3. O projeto simula uma garagem virtual onde os usuários podem visualizar veículos em destaque, navegar pelo pátio de carros, adicionar veículos à garagem e entrar em contato.

## Funcionalidades

- **Página Inicial (index.html)**: Destaques de veículos e notícias sobre o mundo automotivo.
- **Pátio (patio.html)**: Lista completa de veículos disponíveis em formato de cards.
- **Garagem (garagem.html)**: Visualização dos veículos selecionados (simulação de carrinho de compras).
- **Contato (contato.html)**: Formulário para entrar em contato com a concessionária.

## Como Visualizar

1. Clone ou baixe o repositório.
2. Abra qualquer arquivo `.html` (ex: `index.html`) em um navegador web moderno.
3. Navegue pelas páginas usando os links no menu de navegação.

## Estrutura do Projeto

```
garagem/
├── index.html          # Página inicial com destaques e notícias
├── patio.html          # Página com todos os veículos
├── garagem.html        # Página da garagem (carrinho)
├── contato.html        # Página de contato
├── style.css           # Arquivo de estilos CSS
└── img/                # Pasta para imagens (logo, fotos de carros)
    └── logo.jpg        # Logo da Speed Garage
```

## Tecnologias Utilizadas

- **HTML5**: Estrutura das páginas.
- **CSS3**: Estilização responsiva com Flexbox e Media Queries.

## Responsividade

O site é totalmente responsivo, adaptando-se a dispositivos móveis (até 480px), tablets (até 768px) e desktops.
